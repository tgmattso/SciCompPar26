# Scientific Computing in Parallel ... for physics students

These materials support a lecture series on the use of parallel computing for scientific applications. The intended audience is students of Physics.

The repository will hold PDF files for the lectures, a file (learningC.c) that describes all the elements of the C programming language we will use, and a directory of Code.

There are four lectures planned for this series:

* An overview of parallelism and parallel systems (Parallel_systems.pdf)
* An introduction to parallel programming (using OpenMP for CPUs)
* GPU programming using OpenMP with just a bit of CUDA
* MPI overview ... plus closing comments on the future of parallel computing.
